This information that we collect includes:

- <strong>Your Server name where MineBot is registered</strong>
- <strong>Your Discord username</strong>

We may also collect telemetry data to help with developing our application. This is an optional piece of information that you must consent to before we collect any information regarding the actions you make using MineBot.

Telemetry data, is defined as information that is obtained from the actions of a user. This includes, but is not limited to:
- Sending messages to MineBot;
- Interacting directly with the MineBot application in any way; and
- Other actions made through our services.
